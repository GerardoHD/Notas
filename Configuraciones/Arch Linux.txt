#Configuraciones comunes para Arch Linux

Aplicar configuración de Git
Aplicar configuración de GNOME
Aplicar configuración de Linux

Activar soporte AUR en pamac-aur

Firmar bootx64.efi de Brunch:
sudo sbsign --key /etc/efi-keys/db_arch.key --cert /etc/efi-keys/db_arch.pem --output /efi/efi/boot/bootx64.efi /efi/efi/boot/bootx64.efi

Instalar Lenovo Diagnostics a /efi:
mkdir /efi/efi/Lenovo/
cp BOOTX64.efi /efi/efi/Lenovo/Diagnostics.efi
efibootmgr --create --disk /dev/sda --part 1 --label "Lenovo Diagnostics" --loader "\efi\efi\Lenovo\Diagnostics.efi" --verbose
sbsign --key /etc/efi-keys/db_arch.key --cert /etc/efi-keys/db_arch.pem --output /efi/efi/Lenovo/Diagnostics.efi /efi/efi/Lenovo/Diagnostics.efi

Seleccionar carpetas de Disco Duro Externo y copiar a /run/media/gerardohd/Datos/

Configurar Discos de GNOME para que monte la partición de Datos en /run/media/gerardohd/Datos/

Enlazar Carpetas de Medios:
ln -sf /run/media/gerardohd/Datos/GerardoHD/Descargas	/home/gerardohd/Descargas/
ln -sf /run/media/gerardohd/Datos/GerardoHD/Documentos	/home/gerardohd/Documentos/
ln -sf /run/media/gerardohd/Datos/GerardoHD/Imagenes 	/home/gerardohd/Imágenes/
ln -sf /run/media/gerardohd/Datos/GerardoHD/Musica 	/home/gerardohd/Música/
ln -sf /run/media/gerardohd/Datos/GerardoHD/Videos 	/home/gerardohd/Vídeos/
